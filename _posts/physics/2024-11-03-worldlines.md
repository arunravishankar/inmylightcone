A worldline represents the path an object takes through both space and time. Unlike regular paths that show only spatial movement, worldlines include the temporal dimension.

## Key Concepts

1. Timelike Curves
2. Null Geodesics
3. Proper Time

For background, see my post on [curved spacetime](/2024/11/03/curved-spacetime/).
